Analisis_Sentimen.ipynb merupakan notebook Jupyter yang berisi implementasi analisis sentimen menggunakan model RoBERTa. File ini bertugas memproses dan mengklasifikasikan data teks dari platform X (Twitter) untuk mengukur sentimen publik terkait isu ketahanan pangan.

CNN.ipynb adalah notebook Jupyter yang mengimplementasikan Convolutional Neural Network (CNN) untuk tugas klasifikasi gambar. File ini dirancang khusus untuk menilai kualitas visual produk pangan pascapanen, membantu dalam deteksi dini kerusakan atau penurunan mutu.

Clustering GMM.ipynb berisi penerapan algoritma Gaussian Mixture Model (GMM) untuk analisis clustering spasial. File ini digunakan untuk mengelompokkan berbagai wilayah di Indonesia berdasarkan pola dan indikator ketahanan pangan, sehingga dapat mengidentifikasi area yang memerlukan perhatian khusus.

DATASET NUMERIK.xlsx merupakan file spreadsheet yang menyimpan kumpulan data numerik utama proyek. Dataset ini berisi variabel sosio-ekonomi, produksi pertanian, dan indikator ketahanan pangan lainnya yang menjadi fondasi analisis statistik dan pemodelan dalam sistem.

FORECASTING.ipynb adalah notebook Jupyter yang mengimplementasikan model XGBoost untuk peramalan Indeks Ketahanan Pangan (IKP). File ini bertujuan memprediksi tren ketahanan pangan di masa depan berdasarkan data historis dan variabel prediktor.

dataset_teks_analisissentimen_ketahananpangan merupakan file dataset teks mentah yang dikumpulkan khusus untuk keperluan analisis sentimen. Data ini berasal dari platform media sosial dan telah melalui tahap preprocessing awal sebelum diproses lebih lanjut dalam notebook analisis sentimen.
